# 🐄 Cow & Buffalo Image Classification Web App

A deep learning powered web application for classifying cows and buffaloes from images using computer vision and neural networks.

---

## 📌 Project Overview

This project focuses on building an image classification system capable of identifying cows and buffaloes from uploaded images. The application combines deep learning model training with a simple web interface for real-time predictions.

The model is trained using PyTorch and deployed through a Flask-based web application.
---

## 🚀 Features

- Upload cattle images for prediction
- Real-time image classification
- Deep learning based prediction system
- User-friendly web interface
- Trained PyTorch model integration
- Deployment-ready architecture
